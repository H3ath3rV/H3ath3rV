# Heather Vivian

I'm a digital designer and front-end builder working across brand, UI and product design. I enjoy taking an idea beyond the visual stage and turning it into something people can explore and use.

Most people arrive here from my portfolio. This profile is a closer look at the working builds behind those projects, including product prototypes, React interfaces and live websites.

## Selected projects

A local-first desktop app for creating cleaner copies of videos. I designed the interface and built the working processing flow around privacy, clear filter controls and non-destructive output.

**Built with:** React, Flask, Python, FFmpeg and Whisper

**Try it:** [Download the Apple Silicon Mac MVP](https://github.com/H3ath3rV/bleep-bot/releases/latest)

### [Talifun Tokenizer](https://github.com/H3ath3rV/talifun-tokenizer)

A technical product landing page for a high-performance tokenizer, focused on explaining AI infrastructure through clear storytelling, data visualisation and a polished interface.

**Built with:** React, Vite, TypeScript, Tailwind CSS and Framer Motion

**View live:** [h3ath3rv.github.io/talifun-tokenizer](https://h3ath3rv.github.io/talifun-tokenizer/)

### [His Church School](https://github.com/H3ath3rV/his-church-school)

A responsive school website designed around clarity, trust and straightforward parent enquiries, with considered page structure, SEO and accessible forms.

**Built with:** React, Vite, TypeScript, Tailwind CSS, Formspree and GitHub Pages

**View live:** [h3ath3rv.github.io/his-church-school](https://h3ath3rv.github.io/his-church-school/)
